# Lapidary Style Prompt

Short, inscriptional. Best for compressing individual components or small projects.

```
Re-encode the project as a Roman Lapidary Inscription. Use the 'Scriptio Continua' philosophy (minimal formatting) but with spaces. Focus on: Subject + Accomplishment + Tools. Use advanced latin constructions (eg ablative absolute, gerundive etc.) to replace conditional/temporal phrases. Treat lib and lang names as indeclinable. No lists. No pleasantries.
```
