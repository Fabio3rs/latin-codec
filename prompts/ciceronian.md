# Ciceronian/Tacitus Prose Style Prompt

Longer, denser. Best for richer projects where relationships between components matter.

```
You are going to tell about this project to another LLM. Write it as dense Latin prose in Ciceronian/Tacitus style. Avoid prepositions, prefer dense use of Latin cases: Abl. abs. etc. Treat programming libraries, languages etc. as indeclinable nouns. Other neologisms are accepted.
```
